# 🚦 Traffic Sense - Road Safety Awareness Website

A simple and interactive multi-page website designed to spread awareness about traffic rules and road safety. This project allows users to test their knowledge through quizzes, learn important traffic rules, and view their progress after completing the learning modules.

> 📚 This project was built as part of my web development learning journey using HTML, CSS, and JavaScript.

---

## 🌐 Project Overview

Traffic Sense is an educational website that aims to help users understand basic road safety rules in an engaging way. The website includes a pre-test, learning modules, and a final assessment to evaluate the user's understanding.

---

## ✨ Features

- 🏠 Responsive Home Page
- 👤 User Details Form
- 📝 Pre-Test Quiz
- 📚 Traffic Rules Learning Section
- 🏆 Final Assessment
- 📊 Score Comparison
